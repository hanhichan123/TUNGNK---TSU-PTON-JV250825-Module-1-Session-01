What is CSS?

Cascading Style Sheet(CSS) is used to format the layout of the Webpage.

With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or backcground colors are to be used, different displays for different devices anf screen sizes, and much more!

Tip: The word cascading means that a style applied to a parent element will also apply to all children elements within the parent.So, if you set the color of the body text to "blue",all headings, paragraphs, and other text elenments within the body also get the same color(unless you specify something else)!
